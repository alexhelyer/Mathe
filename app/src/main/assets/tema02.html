Números Fraccionarios

También denominados fracciones o quebrados, estos números son una expresión de un cocientes (división) inconcluso en donde se divide un numero entre otro, los números (conjunto a los que pertenece los numero fraccionar (fracciones) a los números racionales.

Números racionales: son los números que pueden ser expresados como una división entre un par de números (un entero y un natural, en otras palabras una fracción).

Representaciones de las fracciones (notación)

  1. Cuando se nombra una fracción el número denominador se toma como partitivo, de manera que al mencionar 1/2 se lee como un medio, 1/3 se lee como un tercio, etc.
  2. Cuando se tiene una fracción negativa simplemente se agrega un signo negativo antes de la fracción, por ejemplo; - 5/7.
  3. Cualquier expresión matemática que se presente como un cociente se le denomina fracción.
  4. Si se tiene una fracción x/y, el producto x por el reciproco de y, es decir x* 1/y, si existe una fracción con ambos números (numerador y denominador) negativos (-x/-y) es puede sobrescribir de la siguiente manera: x/y.

Operaciones con Decimales

Suma:

  • 1/15 + 3/15 = 4/15
  • 4/21 + (-5/21) = -1/21
  • (-5/31) + (-21/31) = -26/31

Resta:

  • 1/15 - 3/15 = -2/15
  • 4/21 - (-5/21) = 9/21
  • (-5/31) - (-21/31) = 16/31

Multiplicación:

  • 1/2 * 5/3 = 5/6
  • 4/7 * (-5/10) = -20/70
  • (-5/21) * (-19/9) = 95/189

División:

  • 2 ÷ 3 = 2/3
  • 1/3 ÷ (-1/2) = -3/2
  • (-9/5) ÷ (-19/9) = 95/81
